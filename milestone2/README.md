# Infosys-Springboard
# Milestone 2 – OTP Authentication & Readability Dashboard

## Description
In this milestone, we enhanced the existing Streamlit application by adding OTP-based authentication, implementing a Readability Dashboard, and improving the overall UI/UX design to provide a more secure and user-friendly experience.

## Features Implemented
- OTP Authentication for secure login/signup verification
- Readability Dashboard with text analysis insights
- Improved UI/UX design and navigation
- Enhanced user interaction flow
- Better layout and visual improvements

## How to Run
1. Install dependencies
2. Run the Streamlit application
3. Open the local URL in your browser
